Q: What are the supported platforms?

Xel works only in environments that support latest Web Platform APIs such as Custom Elements v1, Shadow DOM v1, SVG v2, Pointer Events, CSS variables and ECMAScript 2017:

  • Chrome >= 58
  • Chromium >= 58
  • Opera >= 45
  • Atom Electron >= 1.7
  • NW.js >= 0.23
  • Android WebView >= 58
